# FoodApp
Android application that allows the users to search for any sort of food and view the ingredients.

## Used tools and technologies

### Programming languages: 
    Java 1.8, XML.
### Architucure: 
    MVVM
### Third party libraries
    Retrofit2
    Data binding
    Circle Image view
    Glide
